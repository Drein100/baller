Welcome the Baller.

In this game, you will try to know the name of a randomly generated football player in 8 attempts with clues.

There are 3 modes in the game as easy, medium and hard.

In easy mode, you will start with 3 features.

In medium mode, you will start with 2 features.

In hard mode, you will start with 1 feature.

In each mode, a new feature is unlocked every 3 tries.

Note that you should write both the first and last names of the player.

Good luck!
